# BuildDiff AI Mockup

建設業特化RAGによるPDF差分レビューアプリの静的モックアップです。

## 開き方

`index.html` をブラウザで開いてください。

## 画面構成

- アップロード画面: Before/After資料を投入する初期画面
- レビュー画面: 図面差分、AIレビュー、根拠資料、確認タスク
- サマリ画面: 重要変更、金額影響、関連資料、AI信頼度
- ナレッジ画面: 案件資料を横断検索するRAG画面

## ファイル

- `index.html`: 画面構造
- `styles.css`: UIスタイル
- `app.js`: 画面切替とモック操作
