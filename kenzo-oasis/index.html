OASIS

OASISは、日報、個別進捗管理、GitHub、Slack、Notion、議事録などをもとに、 半期評価・日次フィードバック・アラート確認を行う社内人事評価システムです。

このGitHub Pagesは非機密の入口ページです。社員評価データ、個人情報、APIキー、実スコアは掲載しません。 本体はGoogleログインと許可メール制限のある認証付きWebアプリで運用します。
認証付きOASISへ Private app repository

評価の主軸

  • 業務量: 部署・職種ごとの日々のアウトプット量
  • 積極性: 自発的な動き、前向きな発言、仕事を取りに行く姿勢
  • 再現性: Notionやマニュアルへのナレッジ化

運用方針

  • 正社員、業務委託、アルバイトを区別して評価
  • CompetencyランクはG1からG8で管理
  • 業務委託はSlackオンライン時間ではなく成果物で評価
Limited distribution notice: this page is not an access control layer. Confidential OASIS data must stay behind server-side authentication.
